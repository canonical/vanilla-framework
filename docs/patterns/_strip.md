collection: patterns
title: Strip
---

The strip pattern provides a full width strip container in which to wrap a row.

`.strip` should only contain `.row` and never the other way around.

```html
<div class="strip">
    <div class="row">
        ...
    </div>
</div>
```

<div class="strip">
    <div class="row">
        ...
    </div>
