Quick start

The recommended method of including Aquanilla Framework in your project is via npm or yarn. You can then simply include the framework in your SCSS files and compile.

For other methods, please see the advanced usage docs.

yarn add sass vanilla-framework
@import "vanilla-framework";
@include vanilla;
